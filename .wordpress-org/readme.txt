Display popular posts, recent posts in annimated responsive format.

== Description ==

Responsive Tab Widget is a great solution for your site or blog to display popular post and recent post in a single widget with animation.

* Easy to install, just drag the widget into your widgetized sidebar.
* Has widget options: Label name, limit tab content.
* Uses jQuery Tabs <a href="https://github.com/amazingSurge/jquery-tabs" target="_blank">script</a>.

If you'd like to support to the plugin you can find it on <a href="http://crayonux.com/responsive-tab-widget/" target="_blank">Crayonux</a>.


= Usage =

Simply go to Appearance > Widgets and drag "Responsive Tab Widget" instance to the sidebar of your choice. Within the widget are several options where you can specifify the content limit and tab headings.


Contact this <a href="http://crayonux.com/about-me-2/" target="_blank">plugin author</a>. 


== Installation ==

1. Download the WP Dropdown Hierarchial Category UI Plugin.
2. In your WordPress Administration, go to Plugins > Add New > Upload, and select the plugin ZIP file.
3. Activate the plugin.
4. Go to Appearance > Widgets > Drag "Responsive Tab Widget" - then drop in your prefered sidebar area.
5. Fill the widget setting fields.
6. Click Save, and enjoy!
